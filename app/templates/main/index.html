About Qupiya

What on the Earth is qupiya?

Qupiya is an open-source Password Manager project.

It was created using Flask + Jinja templating system.

Features:

  • ✨ Custom URL token
  • 🙀 Random password Generator
  • 🔐 Goddamn safety
  • 🚀 Profile Page

Plans:

  • Passwords Import
  • Records Log
  • Represent a Table of Passwords

Do I need to Register?

Yep. Logging in is essential part for password managers,

as they need to know whose credentials they're managing.
